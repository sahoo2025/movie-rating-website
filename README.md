# 🎬 Movie Listing and Rating Website

## Project Overview
This project is a responsive Movie Listing and Rating Website built using ReactJS. Users can browse movies, view movie details, and rate movies using a star rating system.

## Features

- Display movies with poster, title, genre, and year
- Star-based movie rating system
- Responsive movie card layout
- Interactive hover effects
- Clean UI design

## Technologies Used

- React.js
- HTML5
- CSS3
- JavaScript
- GitHub Pages (Deployment)
